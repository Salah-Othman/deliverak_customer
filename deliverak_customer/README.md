# 📱 Deliverak Customer Application (تطبيق العميل)

تطبيق الهاتف الخاص بالعميل لنظام **Deliverak** الموجه لخدمة أهالي المنيا وأبو قرقاص. يتيح التطبيق للمستخدمين تصفح المحلات التجارية (مطاعم، سوبر ماركت، صيدليات)، طلب المنتجات، وتتبع الكباتن لحظياً (Real-time Tracking).

---

## 🏗️ الهندسة البرمجية (Architecture)

التطبيق مبني باتباع معمارية **Feature-First Clean Architecture** لضمان سهولة الصيانة والتوسع المستقبلي. ينقسم الكود داخل فولدر `lib/` كالتالي:

```text
lib/
├── core/                  # المكونات المشتركة (تنسيقات الألوان، الثيم، أجهزة المعالجة، والشبكات)
│   ├── theme/             # الألوان والخطوط الخاصة بالبراند
│   ├── network/           # كلاينت Supabase والـ API Layers
│   └── utils/             # دوال مساعدة عامة
│
└── features/              # ميزات التطبيق (كل ميزة جزيرة مستقلة)
    ├── auth/              # ميزة تسجيل الدخول وإنشاء الحساب
    ├── home/              # الشاشة الرئيسية وتصفح التصنيفات والمحلات
    ├── shop_details/      # عرض المنيو والمنتجات والبحث داخل المحل
    ├── cart/              # إدارة السلة محلياً
    └── orders/            # إنشاء الطلبات وتتبع الكابتن لحظياً (Real-time)
